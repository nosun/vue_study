# vue 学习

## vscode 快捷键

- Command + Shift + P / F1 显示命令面板
- Command + Shift + E 左侧浏览器模式
- Command + Shift + X 左侧插件模式
- Command + Shift + F 左侧 Find 模式
- Command + , 打开配置
- Command + Shift + B 调试
